## kn domain delete

Delete a domain mapping

```
kn domain delete NAME
```

### Examples

```

  # Delete domain mappings 'hello.example.com'
  kn domain delete hello.example.com
```

### Options

```
  -h, --help               help for delete
  -n, --namespace string   Specify the namespace to operate in.
```
